HTML & CSS Basics Summary

Html Summary

Html (HyperText Makerup Language) is need to define our page content, structure and meaning. You don't use it for styling purposes. Use Css for that instead!

  1. Html uses "elements" to describe (annotate) content
  2. Html elements typically have an opening teg, content and then a closing tag
  3. You can also have void (empty) elements like umages
  4. You can also configure elements with attributes
  5. There's long list of acailable elements but you'll gain experience over time, no worries.

Learn more about all available HTML elements on the MDN HTML element reference.

CSS Summary

CSS (Cascading Style Sheets) is used for styling your page content.

  1. Style are assigned via property-value pairs
  2. You can assing styles via the "stlye" attribute
  3. To avoid code duplication, you typically use global styles (e.g via the "style" element) instead
  4. alternatively, you can work with external stylesheet files which you "link" to
  5. When working with CSS, concepts like "inheritance", "specificity and the "box model" should be understoood.
  6. Learn more about all available CSS properties and values on the MDN CSS property reference.
